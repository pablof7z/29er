# TUI (29er-tui)

A buildable terminal app target for 29er using Ratatui with a login screen, hierarchical message rendering with per-author colors, composer with @mention popup and publish outbox, command palette, and membership/admin actions — consumes nmp-app-29er as an rlib, not FFI
