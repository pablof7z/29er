# User Settings

## Settings Menu

A gear-icon Settings button appears in the top-right tab bar, replacing the standalone logout button. It opens a menu with the following options:

- `Log Out`
- `Reset Local Database`

<!-- citations: [^019f0-e552d] [^019f0-eaacd] -->
## Reset Local Database

Selecting `Reset Local Database` displays a destructive confirmation dialog. Confirming this action resets the local database and restarts the app. <!-- [^019f0-8ccdc] -->
